# Athar CEO Agent (Separate Folder)

This folder is isolated from your website files.

## Files
- `SKILL.md` -> main CEO agent behavior
- `ceo-coordinator.mdc` -> optional Cursor rule template

## Activate Skill
Use command-style prompts such as:
- `Sab kuch batao`
- `PHYSICS_AGENT: kinematics start karwao`
- `CS_AGENT: C loops assignment solve karwao`
- `WEBDEV_AGENT: landing page improve karo`
- `SOCIAL_AGENT: aaj ka reel caption do`

## Apply Rule (Optional)
If you want this as persistent behavior, copy `ceo-coordinator.mdc` into:
- project rules folder: `.cursor/rules/`
or your personal Cursor rules location, depending on your setup.
