# Jarvis CEO Command Cheatsheet

## 1) System Commands
- `/help` -> command list
- `/status` -> current mode/voice/record settings
- `/time` -> local time
- `/exit` -> close assistant

## 2) Mode Commands
- `/modes` -> all available agent modes
- `/mode PHYSICS_AGENT`
- `/mode WEBDEV_AGENT`
- `/mode SOCIAL_AGENT`
- `/mode auto` -> auto detect mode from prompt

## 3) Voice + Recording Commands
- `/voice on`
- `/voice off`
- `/record 8`
- `/record 12`

## 4) Planning Quick Commands
- `/brief` -> full daily CEO briefing
- `/study` -> academic sprint plan
- `/webdev` -> webdev action list
- `/social` -> social posting action list

## 5) Direct Natural-Language Commands (No slash needed)
- `Sab kuch batao`
- `Physics parao mujhe`
- `MATH_AGENT: differentiation practice karao`
- `CS_AGENT: C loops ka code debug karo`
- `ICT_AGENT: Excel assignment step by step`
- `ENGLISH_AGENT: meri email correct karo`
- `ISLAMIAT_AGENT: topic with references samjhao`
- `PAKISTAN_STUDIES_AGENT: timeline bana do`
- `SCIENCE_AGENT: chapter summary do`
- `WEBDEV_AGENT: meri portfolio site improve karo`
- `SOCIAL_AGENT: aaj ki reel caption + hashtags do`

## 6) Maintenance Commands
- `/clearlog` -> clear `session-log.jsonl`
